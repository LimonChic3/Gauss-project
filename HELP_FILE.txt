GSS hight mu_x mu_y sigma_x sigma_y
Создает Гаусс заданных параметров
Где:
hight – целое число, высота Гаусса
mu_x – целое число, координата вершины Гаусса по х
mu_y – целое число, координата вершины Гаусса по х
sigma_x – целое число, растяжение по х
sigma_y – целое число, растяжение по y

INIT x_size y_size
Создает поле заданного размера
Где:
x_size – целое число, размер поля по x
y_size – целое число, размер поля по y

GEN
Добавляет Гауссы на поле

CRBMP bmp_created_filename.bmp
Создает картинку формата .bmp текущего поля
Где:
bmp_created_filename.bmp – имя файла, в который пользователь хочет записать картинку

RDBMP bmp_read_filename.bmp
Читает картинку формата .bmp в текущее поле
Где:
bmp_read_filename.bmp – имя файла, из которого пользователь хочет прочитать картинку

GNU gnuplot_data_filename.txt Gnuplot_commands_filename.txt
Создает два файла формата .txt. 
Один – с непосредственными командами для гнуплота, второй – с данными (содержание поля), по которым гнуплот будет строить картинку. 
Где:
gnuplot_data_filename.txt – имя файла для записи данных и построения картинки
Gnuplot_commands_filename.txt – имя файла для записи непосредственно команд гнуплота

BIN slice_hight
Обрезает поле по указанному значению
Где:
slice_hight – целое число, высота среза

WAVE min_component_size
Выделяет связные компоненты и отбрасывает слишком мелкие
Где:
component_size – целое число, показывающее, компоненты какого размера мы считаем шумом (мусором) 

KMEANS amount_of_dots
Реализует алгоритм k-средних
Где:
amount_of_dots – целое число для алгоритма k-средних

KMCORE amount_of_cores amount_of_dots_in_cores
Реализует алгоритм k-средних с ядрами
Где:
amount_of_dots – целое число, количество точек в ядре
amount_of_dots_in_cores – целое число, количество ядер
